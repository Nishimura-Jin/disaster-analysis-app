# 災害リスク分析ダッシュボード（日本）

このアプリは、気象庁が公開している警報データをもとに、地域ごとの警報発表の傾向を整理・可視化するためのツールです。

データを保存しながら、地域別や時間帯別の傾向をグラフで確認できます。

---

## ■ このアプリについて

日本全国の気象警報データを取得し、SQLiteに保存したうえで分析しています。

主に以下のような内容を確認できます。

- どの地域で警報が多いか
- どの時間帯に警報が出やすいか
- 警報発表の多い地域ランキング

このアプリは、災害そのものの発生ではなく「警報の発表頻度」をもとに傾向を分析しています。

---

## ■ 使用技術

- Python
- Streamlit
- Pandas
- Matplotlib

---

## ■ データソース

気象庁が公開している防災情報データを使用しています。

https://www.jma.go.jp/bosai/warning/data/warning/map.json

---

## ■ 使い方

### 1. ライブラリのインストール

以下のコマンドで必要なライブラリをインストールしてください。

```bash
pip install streamlit pandas matplotlib requests
```

### 2. アプリ起動

```bash
streamlit run app.py
```

### 3. 操作
アプリを起動したら「データ更新」ボタンを押すことで最新の警報データを取得できます。

取得されたデータは自動で保存され、グラフやランキングとして表示されます。
