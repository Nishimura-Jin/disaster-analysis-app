# 災害リスク分析ツール

気象庁の警報情報と気象データをもとに、都道府県ごとの災害リスクをリアルタイムで可視化・分析するWebアプリです。
過去8日間の実績データと今後7日間の予測データを組み合わせることで、現在の状況把握と今後の備えの両方に活用できます。

## 制作背景

近年、豪雨や猛暑など気象災害のニュースを目にする機会が増えています。今どの地域が危険なのかをリアルタイムで把握し、実際の避難や備えに役立てられるアプリを作りたいと思いこのテーマを選びました。単にデータを表示するだけでなく、過去の実績データと今後の予測を組み合わせることで、現状の把握と今後の見通しを一画面で確認できる設計にしています。

---

## 機能

### 概要タブ
- **最大リスク地域の表示**：全国で最もリスクの高い都道府県とその要因を表示
- **全体分布サマリー**：危険・警戒・注意・安全の県数をメトリクスで表示
- **ドーナツグラフ**：全国の危険度分布を割合で可視化
- **地方別の危険度**：北海道・東北・関東など9ブロック単位で集計した横棒グラフ

### 警報タブ
- **警報情報の取得**：気象庁（JMA）のAPIからリアルタイムの警報・注意報情報を取得
- **警報種別の集計グラフ**：大雨・暴風・大雪など種別ごとに何県で発令中かを横棒グラフで表示

### 地域分析タブ
- **リスクランキング**：全都道府県のリスクスコア・最高気温・降水量を一覧で表示
- **地方×危険度の複合フィルタ**：地方と危険度を組み合わせて絞り込み可能
- **地図表示**：リスクスコアが5以上（注意・警戒・危険）の地域をマップ上に色付きで表示（円の大きさは警報数に対応）

### 気象予測タブ
- **過去8日間の実績**：選択した都道府県の気温・降水量の推移を表示
- **今後7日間の予測**：Open-Meteo予報APIから取得した予測値を同一グラフ上に重ねて表示

## リスクスコアの計算方法

気温・降水量・警報・複合条件の4指標を組み合わせて独自にスコアを算出しています。スコアは最大20点で打ち止めにしています。

### 気温スコア
| 条件 | スコア |
|------|--------|
| 38℃以上 | +5 |
| 35℃以上38℃未満 | +3 |
| 30℃以上35℃未満 | +2 |
| 0℃以下かつ降水あり（凍結・積雪リスク） | +3 |

### 降水スコア
| 条件 | スコア |
|------|--------|
| 50mm以上 | +6 |
| 20mm以上50mm未満 | +3 |

### 複合・警報スコア
| 条件 | スコア |
|------|--------|
| 30℃以上かつ20mm以上（熱中症・土砂災害の複合リスク） | +3 |
| 警報・注意報発令中 | +3（最低スコア5を保証） |

### 判定基準
| スコア | 危険度 |
|--------|--------|
| 15以上 | **危険** |
| 10以上15未満 | **警戒** |
| 5以上10未満 | **注意** |
| 5未満 | **安全** |

- [気象庁 警報・注意報データ](https://www.jma.go.jp/bosai/warning/)

## 実行方法

```bash
# 依存ライブラリのインストール
pip install streamlit pandas requests folium streamlit-folium altair plotly

# アプリの起動
streamlit run app.py
```

起動後、ブラウザで `http://localhost:8501` にアクセスしてください。
初回起動時はデータを自動取得します。2回目以降は「最新データを取得」ボタンで更新できます。

## 工夫した点

- 台風情報のニュースなどで日本地図を使った可視化が直感的でわかりやすいと感じていたため、地図表示を採用しました。どの地域が危険かを一目で把握できるよう、リスクスコアに応じた色分けと、警報数に応じた円の大きさで表現しています。
- 数字やグラフを並べるだけでなく、実際に避難の判断や備えに役立てられる実用的なアプリを目指しました。現在どこで災害が起きているかをリアルタイムで確認できるよう、気象庁の警報情報をそのまま表示する機能を設けています。
- 気温・降水量のデータは時系列の変化を追いやすい折れ線グラフで表示しています。過去8日間の実績と今後7日間の予測を同一グラフ上に重ねることで、現在地と今後の見通しを一画面で比較できるようにしました。
- 気象予測タブのデフォルト表示を大阪に設定しています。自分が大阪在住のため、起動直後から自分の地域のデータをすぐ確認できるようにしました。
